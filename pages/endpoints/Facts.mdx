## Our facts endpoint
Here is a list of how our `/facts` endpoint works and the types of responses you'll get.

<Callout type="info">Want more facts? Join the discord to give suggestions</Callout>

---

### <Code style={{ fontWeight: 400 }}><span style={{color: colors.requests["GET"]}}>GET</span> /facts</Code>

**Example Response:**
```url filename="URL"
https://api.some-random-api.com/facts
```
```json filename="Response"
{
  "endpoints": [
    "/facts/cat",
    "/facts/fox",
    "/facts/birb",
    "/facts/panda",
    "/facts/koala",
    "/facts/kangaroo",
    "/facts/racoon",
    "/facts/giraffe",
    "/facts/whale",
    "/facts/elephant",
    "/facts/dog",
    "/facts/bird",
    "/facts/red_panda"
    ]
  ]
}
```

---

### <Code style={{ fontWeight: 400 }}><span style={{color: colors.requests["GET"]}}>GET</span> /facts/<span style={{color: colors.requests["PARAM"]}}>:type</span></Code>

**Example Response:**
```url filename="URL"
https://api.some-random-api.com/facts/bird
```
```json filename="Response"
{
  "fact": "There are more fake flamingos on Earth than real ones."
}
```
