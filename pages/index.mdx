## Welcome to the SRA Documentation

Some Random API has a numerous amount of endpoints ready to be used withou the use of an API key.

However this does not mean that all the endpoints can be used without one.
Throughout this documentation you will be told whether or not a certain endpoint requires an API key and if so, how to generate one.
