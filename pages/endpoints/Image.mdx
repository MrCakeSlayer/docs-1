## Our images endpoint
Here is a list of how our `/img` endpoint works and the types of responses you'll get.

<Callout type="info">Want more images? Join the discord to give suggestions</Callout>

---

### <Code style={{ fontWeight: 400 }}><span style={{color: colors.requests["GET"]}}>GET</span> /img</Code>

**Example Response:**
```url filename="URL"
https://api.some-random-api.com/img
```
```json filename="Response"
{
  "endpoints": [
    "/img/fox",
    "/img/cat",
    "/img/birb",
    "/img/panda",
    "/img/red_panda",
    "/img/pikachu",
    "/img/racoon",
    "/img/koala",
    "/img/kangaroo",
    "/img/whale",
    "/img/dog",
    "/img/bird"
  ]
}
```

---

### <Code style={{ fontWeight: 400 }}><span style={{color: colors.requests["GET"]}}>GET</span> /img/<span style={{color: colors.requests["PARAM"]}}>:type</span></Code>

**Example Response:**
```url filename="URL"
https://api.some-random-api.com/img/bird
```
```json filename="Response"
{
  "link": "https://i.imgur.com/U5blHCt.jpeg"
}
```
